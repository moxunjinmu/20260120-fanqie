# 列出番茄工作法软件所有的需求

基于之前的深入分析和最新的桌面软件趋势，以下是为开发Windows桌面番茄工作法软件（2026年现代化标准）定制的完整需求列表。

这份需求列表已经为您结构化整理，可以直接作为产品文档（PRD）提供给开发人员或AI编程助手。

### 1. 核心功能需求 (Functional Requirements)

这是软件的“骨架”，必须完整实现。

* **计时器系统 (Timer Logic)**
    * **三种模式**：支持标准的“专注模式”（默认25分钟）、“短休息”（默认5分钟）、“长休息”（默认15分钟）。
    * **自动/手动流转**：
        * 倒计时结束时，自动弹出通知，但**不自动开始**下一阶段（需用户点击确认），防止用户离开时计时器空跑（*行业最佳实践*）。
        * 提供设置选项：允许用户开启“自动开始下一阶段”。
    * **操作控制**：开始、暂停、继续、跳过（放弃当前阶段）、重置。
* **任务管理 (Task Integration)**
    * **待办清单 (To-Do List)**：允许用户添加任务，并为每个任务估算需要的“番茄数”。
    * **当前任务聚焦**：启动计时器时，必须选择一个“当前正在进行的任务”，让时间记录与具体事项关联。
    * **完成勾选**：任务完成后可勾选，移至“已完成”列表。
* **统计与记录 (Stats \& History)**
    * **今日概览**：实时显示今日已完成的专注时长、完成的番茄钟数量。
    * **历史趋势**：简单的柱状图或热力图（类似GitHub Contribution），展示过去7天或30天的专注趋势。


### 2. 交互与体验需求 (UI/UX Requirements)

这是软件的“颜值”和“易用性”，决定用户是否愿意长期使用。

* **系统托盘 (System Tray) - 重点**
    * **后台运行**：点击关闭按钮时，不退出程序，而是最小化到右下角系统托盘。
    * **动态图标**：托盘图标应能通过颜色或微小角标展示当前状态（如：红色代表专注中，绿色代表休息中）。
    * **右键菜单**：提供快捷操作：`开始/暂停`、`跳过`、`退出`，无需打开主界面即可控制。
* **沉浸式体验 (Immersive Experience)**
    * **迷你悬浮窗 (Mini Mode)**：提供一个极简的、半透明的倒计时小条，置顶显示在屏幕边缘，不遮挡工作内容但能随时看到剩余时间。
    * **进度可视化**：在主界面使用圆形进度条（Ring Progress）或波浪动画展示剩余时间，比单纯的数字跳动更具美感且不焦虑。
* **通知与反馈 (Notifications)**
    * **系统原生通知**：阶段结束时发送Windows系统通知（WinUI Toast Notification）。
    * **声音反馈**：
        * 结束提示音（清脆的铃声）。
        * 白噪音支持（可选）：提供“雨声”、“咖啡馆”、“篝火”等背景白噪音，帮助进入心流状态。


### 3. 技术与性能需求 (Technical \& Performance)

这是软件的“内功”，确保软件轻量、稳定。

* **技术栈约束**
    * **框架**：**Tauri 2.0**（确保体积 < 10MB，内存 < 50MB）。
    * **前端**：React 或 Vue 3 + TypeScript。
    * **样式**：Tailwind CSS + Shadcn/UI (或 DaisyUI)，确保实现现代化的圆角和阴影风格。
    * **后端**：Rust (处理系统托盘、文件读写等原生能力)。
* **数据存储**
    * **本地优先**：所有数据（配置、历史记录）存储在本地 JSON 文件或 SQLite 数据库中（使用 `tauri-plugin-store`）。
    * **隐私安全**：无需联网注册登录，即开即用。
* **系统兼容性**
    * **Windows适配**：完美支持 Windows 10 和 Windows 11。
    * **开机自启**：提供“开机自动启动”并“最小化启动”的选项。


### 4. 2026年现代化加分项 (Advanced Features)

如果想让软件脱颖而出，可以考虑这些进阶需求：

* **系统级勿扰 (DND Integration)**：专注阶段开始时，自动触发Windows的“专注助手（Focus Assist）”模式，屏蔽其他软件的通知；休息阶段自动关闭。
* **应用阻断 (App Blocker)**：(可选高级功能) 在专注期间，检测并阻止用户打开黑名单应用（如游戏、社交软件）。
* **Spotify/Music 集成**：专注开始时自动播放指定的“专注歌单”，休息时暂停或切换歌单。


### 总结：给AI的Prompt建议

如果您打算让AI编写代码，建议将上述需求拆解，分批次发送。
**第一步提示词示例**：
> "请使用 Tauri 2.0 + React + Tailwind CSS 初始化一个项目。首先，请实现核心的计时器逻辑（Hook），包含工作25分钟、短休息5分钟、长休息15分钟的状态机切换，暂不需要UI，只需控制台打印状态变化。"

**第二步提示词示例**：
> "现在为刚才的计时器逻辑制作UI。请使用圆形进度条展示倒计时，界面风格模仿 Windows 11 的磨砂玻璃效果（Mica），并添加开始、暂停、跳过的图标按钮。"
